# AI Resume Generator

A one-stop AI-powered resume solution focused on building
ATS-friendly and job-specific resumes.

## Initial Features
- ATS-friendly resume creation
- Job-specific resume tailoring using AI
- Editable AI-generated bullet points
- Clean and minimal resume templates

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- AI: OpenAI API
- Database: MongoDB (planned)

Under active development
